Lógica de Programação

O que é Lógica de Programação?

Lógica de programação é a base de todo desenvolvimento de software. Ela é uma forma de organizar o pensamento para dividir um problema em etapas menores e mais simples, facilitando assim a sua resolução.

Definição: Um conjunto de instruções organizadas em sequência lógica para atingir um objetivo específico.

Exemplo Prático: Cálculo de Média

Um programa que calcula a média de notas de um aluno:

ALGORITMO CalcularMedia
ESCREVA "Digite as notas: "
LER nota1
LER nota2
LER nota3
media = (nota1 + nota2 + nota3) / 3
SE media >= 6 ENTÃO
    ESCREVA "Aprovado"
SENÃO
    ESCREVA "Reprovado"
FIM

Importância para os Programadores

1. Resolução de Problemas

Dividir um problema em partes menores facilita o entendimento e encontrar a solução. Ao quebrar a complexidade em componentes simples, o caminho até a resposta fica mais claro.

2. Eficiência

Algoritmos bem planejados consomem menos tempo de processamento e recursos do computador. Uma boa lógica é fundamental para criar programas rápidos e responsivos.

3. Facilidade de Manutenção

Código com boa lógica é mais fácil de atualizar e corrigir. Quando outra pessoa (ou você mesmo meses depois) precisar fazer alterações, um código bem estruturado é muito mais compreensível.

Importância no Dia a Dia

Tomada de Decisões

Usamos lógica para planejar, definir prioridades e criar estratégias. Exemplo: planejar uma viagem envolve lógica ao decidir rotas, datas e recursos necessários.

Comunicação

O raciocínio lógico ajuda a formular respostas coerentes e se expressar com clareza. Uma comunicação bem estruturada segue uma lógica clara do pensamento.

Solução de Problemas

Aplicamos lógica ao analisar e resolver situações cotidianas, desde problemas simples até situações complexas que exigem pensamento analítico.

Atividades - Pseudocódigo

Exercício 1

Elabore um algoritmo que leia dois números e mostre a soma deles.

Exercício 2

Faça um pseudocódigo de um algoritmo que receba 4 notas, calcule e mostre a média aritmética entre elas.

Exercício 3

Escreva um algoritmo que leia uma medida em metros e converta para centímetros.

Conceitos-Chave

Conceito Descrição Exemplo
Sequência Instruções executadas em ordem Passo 1 → Passo 2 → Passo 3
Seleção Escolher caminho baseado em condição SE nota >= 6 ENTÃO aprovado
Repetição Executar instruções múltiplas vezes ENQUANTO i <= 10 FAÇA
Variável Guardar valores em memória idade = 25
Operador Realizar operações matemáticas +, -, *, /, %
